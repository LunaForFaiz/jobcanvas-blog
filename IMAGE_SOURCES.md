## Sourcing Options

### Option 1: Unsplash/Pexels (Free Stock)
- Search terms: "resume optimization," "job interview," "data analysis charts"
- Download high-res versions
- Place in `/images/` directory

### Option 2: Generate with AI  
- Use DALL-E, Midjourney, or Stable Diffusion
- Prompts for charts/graphs work best
- Professional, clean aesthetic

### Option 3: Create Custom Charts
- Use Canva or Google Sheets for data visualizations
- Export as PNG or JPG
- Ensure mobile-responsive sizing (1200x630px ideal for hero images)
